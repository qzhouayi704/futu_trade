# 简化版富途量化交易系统

一个高度简化的富途量化交易系统，专注于核心功能的实现。

## 功能特性

1. **股票池数据初始化** - 从富途API或模拟数据初始化股票池
2. **股票池实时报价** - 实时获取和显示股票价格信息
3. **交易预警** - 价格变化和成交量异常预警
4. **交易条件滚动显示** - 实时显示触发的交易条件
5. **自动交易** - 根据预警条件自动生成交易信号

## 系统架构

```
simple_trade/
├── simple_trade.py       # 主程序（包含所有后端逻辑）
├── config.json          # 配置文件
├── requirements.txt     # Python依赖
├── templates/
│   └── index.html      # 前端界面
├── static/
│   ├── style.css       # 样式文件
│   └── app.js          # 前端脚本
└── data/
    ├── trade.db        # SQLite数据库（自动创建）
    └── system.log      # 系统日志（自动创建）
```

## 快速开始

### 1. 安装依赖

```bash
pip install -r requirements.txt
```

### 2. 配置系统

编辑 `config.json` 文件：

```json
{
  "futu_host": "127.0.0.1",          # 富途OpenD地址
  "futu_port": 11111,                # 富途OpenD端口
  "database_path": "data/trade.db",  # 数据库路径
  "update_interval": 5,              # 更新间隔（秒）
  "auto_trade": false,               # 是否自动交易
  "price_change_threshold": 3.0,     # 价格变化预警阈值（%）
  "volume_surge_threshold": 2.0,     # 成交量异常阈值
  "max_stocks_monitor": 50           # 最大监控股票数量
}
```

### 3. 启动系统

```bash
python simple_trade.py
```

或指定参数：

```bash
python simple_trade.py --host 0.0.0.0 --port 5000 --debug
```

### 4. 访问界面

打开浏览器访问：`http://localhost:5000`

## 使用说明

### 初始化股票池

1. 点击 "初始化股票池" 按钮
2. 系统会从富途API获取板块和股票数据（如果富途未连接则使用模拟数据）
3. 初始化完成后会显示板块和股票数量

### 启动监控

1. 点击 "启动监控" 按钮
2. 系统开始定时获取股票实时报价
3. 实时更新股票价格、检查预警条件、生成交易信号

### 查看数据

- **股票池实时报价**: 显示所有活跃股票的实时价格信息
- **交易预警**: 显示价格变化超过阈值的预警信息
- **交易条件**: 显示最近触发的交易条件和信号

## 配置选项

| 参数 | 说明 | 默认值 |
|------|------|--------|
| futu_host | 富途OpenD地址 | 127.0.0.1 |
| futu_port | 富途OpenD端口 | 11111 |
| update_interval | 数据更新间隔（秒） | 5 |
| auto_trade | 是否启用自动交易 | false |
| price_change_threshold | 价格变化预警阈值（%） | 3.0 |
| volume_surge_threshold | 成交量异常阈值倍数 | 2.0 |
| max_stocks_monitor | 最大监控股票数量 | 50 |

## 数据库结构

系统使用SQLite数据库，包含4个核心表：

- **plates** - 板块信息表
- **stocks** - 股票池表
- **trade_signals** - 交易信号表
- **system_config** - 系统配置表

## API接口

| 接口 | 方法 | 说明 |
|------|------|------|
| `/api/init` | POST | 初始化股票池 |
| `/api/quotes` | GET | 获取实时报价 |
| `/api/alerts` | GET | 获取交易预警 |
| `/api/conditions` | GET | 获取交易条件 |
| `/api/status` | GET | 获取系统状态 |
| `/api/start` | POST | 启动监控 |
| `/api/stop` | POST | 停止监控 |

## 富途API配置

如果要使用富途真实数据，需要：

1. 安装富途OpenD客户端
2. 启动OpenD并登录
3. 确保OpenD监听地址为 `127.0.0.1:11111`
4. 如果富途API不可用，系统会自动使用模拟数据

## 模拟数据

系统内置模拟数据功能，包含：

- 6个模拟板块（港股银行、地产、科技、医药，美股科技、金融）
- 18只模拟股票（包含知名港股和美股）
- 随机生成的价格和成交量数据

## 系统特点

1. **极简架构** - 单文件后端，所有功能集中在一个Python文件中
2. **零配置运行** - 开箱即用，支持模拟数据
3. **实时更新** - 基于WebSocket的实时数据推送
4. **响应式界面** - 基于Bootstrap的现代化界面
5. **完整功能** - 涵盖股票池、报价、预警、交易的完整流程

## 注意事项

1. 本系统为简化版本，仅用于学习和演示
2. 自动交易功能目前只生成交易信号，不执行真实交易
3. 生产环境使用需要增加更多的安全和错误处理机制
4. 富途API使用需要遵循相关的使用条款和限制

## 故障排除

### 富途连接失败
- 检查富途OpenD是否正在运行
- 确认IP地址和端口配置正确
- 检查防火墙设置

### 数据库错误
- 确保data目录存在且有写入权限
- 删除data/trade.db文件重新初始化

### 端口占用
- 使用 `--port` 参数指定其他端口
- 检查5000端口是否被其他程序占用
